¿Que es GIT Bash?

GIT Bahs es una aplicaciion que proporciona una linea de comandos para interactuar GIT en un Sistema Operativo Windows. Es una emulacion de la terminal Unix que permite a los usuarios ejecutar comandos GIT

¿Como dar seguimiento a un archivo de texto con GIT?

primero que todo tenemos que crear un archivo de texto en una carpeta ya creada previamente y colocarle un nombre como por ejemplo: programacion, ahora hacemos clic derecho en la carpeta ya abierta y cliceamos en el apartado crear documento de texto y colocamos cualquier nombre, en este caso GIT

Luego abrimos nuestra aplicacion de git bash, y al abrirlo colocar los siguientes codigos:

1) git config --global user.name "nombre"

2) git config --global user.email "correo electronico"

git: hace referencia al programa

config:indica a git que vamos a modificar un elemento de configuracion.

--global: el elemento de configuracion se va a cambiar de forma global (en toda la computadora) y se usará como valor por defecto cuando no exista de forma loca.
