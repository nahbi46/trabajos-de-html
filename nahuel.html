primero que todo tenemos que crear un archivo de texto en una carpeta ya creada previamente y colocarle un nombre como por ejemplo: programacion, ahora hacemos clic derecho en la carpeta ya abierta y cliceamos en el apartado crear documento de texto y colocamos cualquier nombre, en este caso GIT

Luego abrimos nuestra aplicacion de git bash, y al abrirlo colocar los siguientes codigos:

1) git config --global user.name "nombre"

2) git config --global user.email "correo electronico"
